## Social Media Project  
  An social media website for people.
</div>

## **How to install and run locally ?**

```
$ git clone https://github.com/VinayJv/social-media-project-2.git
$ npm install
$ npm start
```
## **Features -**

- User Feed
- List all user posts
- Sort by Date & Trending
- Create a Post
- Like a post
- Edit & Delete Post
- User Profile
- Add Avatar
- Add Bio
- Add Portfolio URL
- Follow/Unfollow
- Explore Feed
- Bookmark Post
- Authentication
- Sign-up
- Login
- Logout

## **Built with -**

- React JS
- React Context API + useReducer
- React Router v6
- React Toastify
- React Device Detect
